# 🌱 PvZRSkinPicker

> A mod for *Plants vs. Zombies™: Replanted* that allows you to select skins for plants and zombies directly from the **Almanac**.
>
> The skin picker is accessible both from the **main menu** and the **pause menu** during gameplay.

> [!WARNING]
> **This mod does NOT unlock pre-order content.**
>
> 🔒 The pre-orde skins will **only appear in the skin picker if the player already owns it**.

## 📦 Download

- [GameBanana](https://gamebanana.com/mods/644298)

## 🧩 Requirements

* 🎮 A copy of **Plants vs. Zombies™: Replanted**.
* 🍉 **[MelonLoader](https://github.com/LavaGang/MelonLoader)** - required mod loader.

## ✨ Features

* 🎭 Pick specific skins for plants and zombies.
* 💾 Selection persistence.
* 🎨 [Custom plant skins](./CustomSkinPacks.md).
* 🚧 ...

## 🐛 Vanilla bug fixes

* Fixes the Retro Peashooter bug where incorrect projectiles and visual effects are used if it is toggled mid game.
* Fixes an incompatibility of deserialized December zombies with retro zombie toggle (exception & wrong drops).

## 🚀 How to use

1. **Install MelonLoader**

    > [Download here](https://github.com/LavaGang/MelonLoader/releases)

2. **Place the Mod DLL**

    > Copy `PvZRSkinPicker.dll` file into the game’s `Mods` folder.

## 🧩 Supported skins

* 🌱 Default
* 🎁 Pre-order/Retro **(only if owned)**
* 🖥️ Platform
* 🌏 China
* 🥚 EasterEgg
* 🎄 Xmas
* 🎨 Custom
